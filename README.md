# Tourism Management System

Spring Boot приложение для управления туристическими турами с PostgreSQL и расширенными бизнес-операциями.

## Тема проекта
**Система управления туристическими турами** - платформа для организации, бронирования и отзывов о турах.
##  Основные сущности
- **Destination** - направления/локации (Париж, Рим, Бали и т.д.)
- **Guide** - гиды (специализация, опыт, контакты)
- **Tour** - туры (даты, цена, доступные места, статус)
- **Booking** - бронирования (клиенты, статусы, транзакции)
- **Review** - отзывы (рейтинги, проверка участия)

##  Технологии
- **Java 17 + Spring Boot 3.5.5**
- **PostgreSQL** - реляционная база данных
- **Spring Data JPA** - работа с данными
- **Flyway** - миграции базы данных
- **Maven** - сборка проекта
- **REST API** - полный CRUD + бизнес-операции
